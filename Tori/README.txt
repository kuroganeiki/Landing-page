プロジェクト名：練習用ランディングページ
====================

【概要】
このサイトは、HTML・CSS・JavaScriptの勉強のために作りました。

実際のWebサイトを参考にして、レイアウトやデザインをまねしながら作りました。

【使用技術】
・HTML
・CSS
・JavaScript
・Bootstrap

【主な内容】

1. レスポンシブ対応
   スマートフォンやパソコンでも見やすいように作りました。

2. ナビゲーションバー
   ドロップダウンメニューを使っています。

3. イベントセクション
   カードデザインでイベント情報を表示しています。

4. 予約フォーム
   名前や人数を入力して、ボタンを押すとメッセージが表示されます。
   また、未入力の場合はエラーを表示します。

【目的】
Web制作の基本を理解するために作りました。
これからもJavaScriptなどを勉強して、スキルを上げたいです。
